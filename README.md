## 介绍

RCLiveVideoLib是融云 为用户提供了一套便捷并且安全稳定视频直播 SDK。用户可以：
* 两行代码开播
* 快速切换常用连麦模式
* 处理连麦用户点击事件

## 安装

推荐使用Pod安装：
```
pod 'RCLiveVideoLib', '~> 0.0.1'
```
